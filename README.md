# SUBMARINUS

A little game about piloting a submarine through a series of caverns.
Not playable yet!

It's programmed in Rust, using the game engine Bevy.

## Inspiration

It's loosely based on a TRS-80 game whose name I forgot.

Some of the game design elements (for example no saves, just do the
best you can until you die) come directly from that 80s style of games.

# How to play

Instructions for Linux
```
git clone https://github.com/amtep/submarinus
cd submarinus
cargo run
```

At this point there's no quit option yet, so you'll have to kill the game with Alt-F4 to quit.

If you don't have rust and cargo yet, see the Rust
[get started](https://rust-lang.org/learn/get-started) page.
